# Customer-Behavior-Analysis
This repository contains the analysis and outcomes of a customer behavior study. The project aims to understand customer preferences, purchasing patterns, and other behavioral insights to drive data-driven decision-making. The analysis is performed using Python, MySQL, and data visualization libraries like Matplotlib and Seaborn.

## Overview

The project involves connecting to a MySQL database, processing customer data, and generating insights through visualizations. Key areas of analysis include:

1. **Customer Journey Analysis**: Identifying drop-off points and average duration at each stage of the customer journey.
2. **Product Ratings**: Analyzing the highest-rated and lowest-rated products.
3. **Regional Sales Performance**: Identifying best-performing products by region and country-wise sales distribution.
4. **Customer Demographics**: Analyzing gender distribution across countries.
5. **Engagement Analysis**: Identifying the most engaging content type and most liked products in marketing campaigns.
6. **Customer Retention**: Comparing repeat vs. first-time buyers and calculating customer retention rates.

## Tools and Technologies

- **Programming Language**: Python
- **Database**: MySQL
- **Libraries**:
  - pymysql: For connecting to and interacting with the MySQL database.
  - pandas: For data manipulation and analysis.
  - matplotlib and seaborn: For data visualization.
- **Data Source**: CSV files containing customer behavior data.
